‣ ntopng is a network traffic probe that provides 360° Network visibility, with its ability to gather traffic information from traffic mirrors, Netflow exporters, SNMP devices, Firewall logs, Intrusion Detection systems.
‣ In a home network context, it can be used to detect P2P activity, malicious activities, live traffic performance analytics when testing bonding solutions such as retransmissions.
‣ Deep packet inspection to identify encrypted traffic type.
Setup on machines with more than 2GB storage free space is done by simply clicking Download and Enable in Configuration tab above.
Setup on embedded routers with less than 4GB storage utilize external storage:
Download and flash image via dd/BalenaEtcher from: https://github.com/SmoothWAN/SmoothWAN-chroot-imagebuilder/releases/tag/main

‣ Slate AX / Flint: smoothwan_debuster_armv7l.img.gz
‣ Raspberry Pi 4 / Banana Pi R64/R3: smoothwan_debuster_aarch64.img.gz
‣ PC-Intel/AMD: smoothwan_debuster_x86_64.img.gz

Then simply plug-in the storage medium via USB, and check Enable in Configuration tab, Save & Apply.
