‣ Tailscale is a zero config VPN for building secure networks.
‣ Easily access your LAN network remotely.
Setup after installing Speedify or setting up a WAN:
Log result should be similiar to:
Visit: http://172.17.17.2:8088/ to login to Tailscale.
After a succesful login, visit: https://login.tailscale.com/admin/machines and toggle settings as following:



Done! You should be able to access your local (LAN) devices from range 172.17.17.0 to 172.17.17.255 remotely on any device that has Tailscale connected to the same account.

Notes:

‣ You can also use Tailscale auto-assigned IP address.
‣ The example assumes that the LAN IP is 172.17.17.2
‣ Reboot SmoothWAN if you have changed the LAN address and use the new address instead.
‣ You can enable "exit node" option in dashboard to use your home networking internet on the go. E.g Bypass Netflix location / account sharing restrictions.
